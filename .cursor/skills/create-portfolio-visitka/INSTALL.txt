ГЕНЕРАТОР ПОРТФОЛИО — Установка в Cursor
=========================================

1. Распакуйте эту папку В папку навыков Cursor, чтобы получилось:
   create-portfolio-visitka/
   ├── SKILL.md
   ├── template.html
   ├── README.md
   └── INSTALL.txt

   Где лежит папка навыков:
   • Для всех проектов:  ~/.cursor/skills/
   • Для одного проекта: <ваш-проект>/.cursor/skills/

2. Перезапустите Cursor или откройте новый чат.

3. В чате напишите, например:
   • «Сделай визитку» / «Создай портфолио»
   • «Сделай визитку с моими данными» (Cursor спросит ваши данные)

4. Получите один index.html — откройте в браузере или залейте на хостинг.

Готово. Та же страница портфолио, ваши данные.
