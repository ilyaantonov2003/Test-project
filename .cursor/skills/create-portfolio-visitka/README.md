# Генератор портфолио

**Навык для Cursor**: создаёт одну и ту же одностраничную визитку для любого пользователя. Один HTML-файл, без зависимостей, с анимациями и адаптивной вёрсткой.

**Как поделиться с друзьями:** передайте им zip-архив. Они распаковывают его в папку навыков Cursor и могут сгенерировать такую же страницу портфолио для себя (со своим именем, описанием, ссылками и т.д.).

---

## Установка (для друзей)

1. **Распакуйте** эту папку в папку навыков Cursor:
   - **Для всех проектов:** распакуйте в `~/.cursor/skills/`  
     Должна появиться папка `~/.cursor/skills/create-portfolio-visitka/` с файлами внутри.
   - **Только для одного проекта:** распакуйте в `.cursor/skills/` в корне проекта.
2. **Перезапустите Cursor** или откройте новый чат, чтобы навык подгрузился.

---

## Как пользоваться (сгенерировать своё портфолио)

В чате Cursor напишите, например:

- **«Сделай визитку»** / **«Создай страницу портфолио»**  
  → Cursor создаст `index.html` с плейсхолдерами. Замените имя, должность, описание, текст «О себе», email и ссылки на соцсети на свои.

- **«Сделай визитку с моими данными»** / **«Создай портфолио и подставь мои данные»**  
  → Cursor спросит имя, должность, краткое описание, абзацы «О себе», email и ссылки, затем сгенерирует `index.html` с вашими данными.

Готовый `index.html` откройте в браузере или залейте на любой хостинг.

---

## Что внутри

- **SKILL.md** — инструкции для Cursor (не удаляйте).
- **template.html** — эталонная вёрстка и стили (только плейсхолдеры).
- **README.md** — этот файл.
- **INSTALL.txt** — краткая инструкция по установке (для тех, кто только что распаковал архив).
